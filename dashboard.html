Help Center

Admin Panel Features

Homework Management: Create, edit, and manage homework assignments for all classes.

Announcements: Send important updates to students, parents, and teachers.

User Management: Add, edit, and manage user accounts (students, teachers, parents).

Messages: View and respond to messages from users.

Analytics: Monitor system usage, homework completion rates, and user activity.

Quick Tips

  • Use the filter tabs to quickly find specific homework or announcements
  • Always double-check the target audience when sending announcements
  • Regularly monitor the analytics to understand user engagement
  • Respond to help requests within 24 hours for best user experience
